                    Risk Assessment Objek Wisata

Semua file dokumen (Word, Excel, PDF) telah dikonversi ke format teks (.txt)
agar dapat dibaca oleh Cursor dan software lainnya.

================================================================================
FILE DOKUMEN UTAMA
================================================================================

1. SPESIFIKASI DATA OBWIS.txt
   - Spesifikasi data untuk Objek Wisata
   - Berisi struktur data yang diperlukan

2. DATA OBWIS.txt
   - Data Objek Wisata

3. Petunjuk pengisian check list wahana.txt
   - Petunjuk pengisian nilai kriteria:
     * 0 = Tidak dapat dipenuhi (harus isi temuan dan rekomendasi)
     * 1 = Terdapat kekurangan (harus isi temuan dan rekomendasi)
     * 2 = Dapat dipenuhi (tidak ada temuan, tidak perlu rekomendasi)
   - Pengisian referensi: bukti surat, dokumentasi foto

4. RISK ASSESMENT OBJEK WISATA 2025.txt
   - File Excel yang berisi struktur penilaian
   - Terdiri dari beberapa sheet:
     * INFRASTRUKTUR
     * KEAMANAN
     * KESELAMATAN
     * KESEHATAN
     * (dan lainnya)

================================================================================
STRUKTUR PENILAIAN
================================================================================

Berdasarkan file RISK ASSESMENT OBJEK WISATA 2025.txt, struktur penilaian:

1. INFRASTRUKTUR
   - A. KELAIKAN GEDUNG/VENUE (Bobot: 0.12)
   - B. KELENGKAPAN HOTEL (Bobot: 0.08)

2. KEAMANAN (Bobot Elemen: 0.2)
   - A. PROSEDUR PELAKSANA PENGAMANAN
   - B. PELAYANAN KEAMANAN

3. KESELAMATAN (Bobot Elemen: 0.25)
   - A. PELAYANAN SISTEM KESELAMATAN
   - B. PETUGAS KESELAMATAN

4. KESEHATAN (Bobot Elemen: 0.1)
   - AB. PROSEDUR PETUGAS MEDIS
   - B. PELAYANAN KESEHATAN

Setiap kriteria memiliki:
- Bobot Elemen
- Nomor Kriteria
- Deskripsi Kriteria
- Pencapaian Nilai Kriteria (0, 1, atau 2)
- Pencapaian Nilai Elemen
- Referensi Dokumen/Rekaman
- Temuan
- Rekomendasi
